## 功能点备注

### 页面样式调整

- [ ] 翻译方向，鼠标滑过自动展现，不用点一下再出来了
- [ ] textarea 输入输出焦点框去掉，或者使用其他方式实现
- [ ] 翻译源 tab 鼠标滑过有获的焦点的效果，但不用自动切换

### 翻译自动语言识别

- 自动：有中文则 自动-英文，否则 自动-中文
- 自动-中文：auto-zhcn
- 自动-英文：auto-en
- 语言-语言：特定语言到特定语言，预先自定义（待定）
